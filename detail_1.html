Cara Mengatasi HP Android Yang Lemot

Pada video kali ini saya ingin share tutorial cara mengatasi HP Android yang suka lemot.

In this video I want to share a tutorial on how to deal with Android phones that are slow.

Caranya cukup mudah, kalian tinggal ke Pengaturan di HP Android kalian, lalu kalian cari Storage dan pilih Bersihkan Memori sampai muncul tulisan. Lalu jika sudah, kalian pilih Penyimpanan, lalu kalian pilih Cache, dan jika muncul peringatan, pilih Ya.

The way is easy, you just go to Settings on your Android phone, then you look for Storage and select Clean Memory until the text appears. Then if so, you choose Storage, then you choose Cache, and if a warning appears, select Yes.

Maka, jika sudah berhasil, maka Cache memory yang ada di HP Android kalian sudah tidak ada kategori Cache memory (kalaupun masih ada Cache memory-nya kalian tinggal ulangi tahapnya lagi) dan kalian telah berhasil menghapusnya dan InsyaAllah bisa ringan lagi performa smartphone kalian.

So, if it has succeeded, then the Cache memory in your Android cellphone is no longer in the Cache memory category (even if there is still Cache memory, you just need to repeat the stage again) and you have successfully deleted it and God willing, it can lighten the performance of your smartphone again.
