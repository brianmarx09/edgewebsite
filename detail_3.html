Cara Menghapus Dual Boot OS Linux dengan EasyBCD

Pada video ini saya memberikan tutorial cara menghapus Dual Boot sistem operasi pada Linux dan Windows.

In this video I want to share a tutorial on how to remove Dual Boot operating systems on Linux and Windows.

Distro Linux yang saya hapus dual disini adalah Lubuntu 16.04 (LTS). Caranya dengan menggunakan software EasyBCD, bertujuan agar tidak terjadi error pada grub rescue partition pada laptop kalian, sehingga kita tidak perlu lagi untuk membuat bootable flash di USB.

The Linux distro that I dual removed here is Lubuntu 16.04 (LTS). You do this by using EasyBCD software, aiming to prevent errors in the Grub rescue partition on your laptop, so that we no longer need to make a bootable flash on USB.

Maka, jika kalian sudah mengunduh softwarenya, kalian tinggal install saja softwarenya (agar lebih jelas panduan penginstallannya diharapkan kalian lihat saja pada video tutorial yang saya berikan, agar kalian lebih mengerti cara penginstallan pada software EasyBCD-nya), jika kalian sudah menginstall softwarenya, tahap selanjutnya adalah kalian buka software EasyBCD-nya lalu kalian cari BCD Deployment pada kategori EasyBCD Toolbox disebelah kiri dan klik BCD Deployment-nya.

So, if you have downloaded the software, all you have to do is install the software (so that the installation guide is clearer, hopefully you can just look at the video tutorial that I provide, so that you better understand how to install the EasyBCD software), if you have installed the software, step next is you open the EasyBCD software then you look for BCD Deployment in the EasyBCD Toolbox category on the left and click on the BCD Deployment.

Selanjutnya jika kalian sudah mengklik kategori BCD Deployment, kalian klik tombol Write MBR, fungsinya itu untuk membuat MBR pada OS urutan Pertama kalian, yaitu Microsoft Windows jika kalian pernah membuat dual boot OS dengan Linux.

Next, if you have clicked on the BCD Deployment category, you click the Write MBR button, its function is to create an MBR on your first OS, namely Microsoft Windows if you have ever made a dual boot OS with Linux.

Tahap terakhir, jika kalian sudah mengklik tombol Write MBR pada BCD Deployment-nya, kalian tinggal restart PC/Laptop kalian agar kalian bisa melihat efeknya setelah kalian menulis ulang MBR pada OS PC/Laptop kalian.

The last stage, if you have clicked the Write MBR button on the BCD Deployment, you just need to restart your PC / Laptop so you can see the effect after you rewrite the MBR on your PC / Laptop OS. Jika kalian masih bingung dengan tutorial yang disini, kalian juga bisa tonton videonya disini
